## Installation

1. Clone the repository 
```sh
git clone https://github.com/Coucat/Pokedex.git
```
2. Change directory
```sh
cd pokedex
```
3. Install NPM packages
```sh
npm install
```
* Or
```sh
yarn install
```
4. Run project
```sh
npm start
```
* Or
```sh
yarn start
```
