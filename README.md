## Installation
1.  Get the link of project at [Coucat](https://github.com/Coucat/Pokedex)
2. Clone the repository 
```sh
git clone https://github.com/Coucat/Pokedex.git
```
3. Install NPM packages
```sh
npm install
```
4. Run project
```sh
npm start
```
